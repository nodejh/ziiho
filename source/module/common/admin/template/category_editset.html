  • 在网站导航显示:
    {eval $index=0;/} {loop yesno_val() $val} {eval $index++;/}  {field:val.title/} {/loop}
    设为在网站导航显示
  • 说明:
    主要用于叙述分类的介绍，不支持html代码

  • SEO设置

  • title 标题:
    设置当前分类的title 标题，将会作为当前分类要显示在网站的标题，不支持html代码
  • keywords 关键字:

    设置当前分类的keywords 关键字，将会作为当前分类的关键字，不支持html代码

    可用变量如下:

    {title}:分类名称

  • description 内容:

    设置当前分类的description 内容，将会作为当前分类的介绍内容，不支持html代码

    可用变量如下:

    {title}:分类名称
